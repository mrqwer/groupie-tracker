# Alem01 - Groupie-Tracker Project

A project aims to manupulate and represent data given by the API on the user-friendly interface 


## API Reference

#### Get all artists

```bash
  GET localhost:8080/
```

#### Get specific artist

```bash
  GET /artist/{id}
```

## Usage
On the root folder of the project, run:
```bash
    go run main.go
```
